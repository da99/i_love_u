Installation and Usage:
-----------------------

    your_shell> npm install i_love_u

    # In your coffeescript code:

    love = require("i_love_u")

    u = new love.i_love_u """
      One is: 1.
      Six is: One + 5.
    """
    u.run()
    u.list() // Instead of 'stack', I call it a 'list'.
    
    # --> ["1", 6.0]
