
i\_love\_u
---------

Fall in love with the computer all over again.

  
Installation and Usage:
-----------------------

    your_shell> npm install i_love_u

    # In your coffeescript code:

    love = require("i_love_u")

    u = new love.Code """
      Val is 1.
      Val + 5.
    """
    u.run()
    u.stack()
    
    # --> ["1", 6.0]
